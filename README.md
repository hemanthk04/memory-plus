# Memory Plus

An MCP-compatible AI knowledge engine that provides long-term memory for AI assistants.

## Tech Stack

- Hono
- TypeScript
- Drizzle ORM
- PostgreSQL
- Supabase
- OpenAI Embeddings (planned)
